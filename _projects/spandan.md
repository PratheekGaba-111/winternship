## Project Overview

Spandan is an interactive classroom platform that transforms live teaching sessions into engaging learning experiences. The system captures teacher's speech in real-time, converts it into transcripts, automatically generates relevant questions from the content, and presents them to students through a dedicated portal for instant interaction and assessment.

## Key Features

- Real-time speech-to-text transcription
- AI-powered question generation from transcripts
- Student portal for answering questions
- Live interaction during classes
- Question and answer tracking
- Teacher dashboard for monitoring student responses

## Technologies Used

- Frontend: React, TypeScript
- Backend: Express.js, Node.js
- Database: MongoDB
- Speech Recognition: OpenAI Whisper, ASR
- Question Generation: Open Source LLM models

## Project Goals

1. Enable seamless real-time classroom interaction
2. Automate question generation from live lectures
3. Provide instant student engagement and feedback
4. Track student participation and understanding
5. Create accessible transcripts for all sessions
